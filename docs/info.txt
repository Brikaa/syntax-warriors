Syntax Warriors is a full-stack web application for programming contests.
Users submit solutions to contests, their code is executed using a code execution engine
and the execution result is checked against a pre-defined set of inputs and outputs.

The technologies that are used are: NodeJS, MySQL, ExpressJS
